## Quick Actions

| Action | Steps |
|--------|-------|
| **Add Supplier** | Click "+ Add Supplier" → Fill form → Submit |
| **View Supplier** | Click supplier name in list |
| **Edit Supplier** | Click edit icon → Modify → Update |
| **Delete Supplier** | Click delete icon → Confirm |
| **Search** | Type in search box |
| **Filter** | Select status dropdown |
| **Export All** | Click "Export" button |
| **Export One** | View supplier → Click "Export" |
| **Print Report** | View supplier → Click "Print" |
